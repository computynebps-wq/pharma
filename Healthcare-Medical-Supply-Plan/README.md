# Medical-Supply-Plan

A git repository for the **Medical Supply Chain Blockchain** initiative: ideas, references, MVP documentation, and sample/template code for a blockchain-based medical supply chain (drug registration, licensing, unit tracking, and verification).

---

## What is this repository?

This repository holds the full plan and artefacts for a **Medical Supply Chain Blockchain** project. It covers:

- **Concept and background** — Problem statement, blockchain in healthcare, and references (on `master`).
- **MVP documentation** — Executive summary, PRD, technical architecture, smart contract spec, data model, wireframes, NFRs, timeline, legal/risk notes (on `docs`).
- **Code** — Template and sample MVP projects (e.g. Hyperledger Fabric chaincode, API, frontend) (on `code`).

Content is organised by **branch**: each branch has a clear purpose so you can work with ideas, docs, or code independently.

---

## Branch overview

| Branch     | Purpose                   | Contents                                                                                                                                                                                                                                                         |
| ---------- | ------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **master** | Concept and references    | Idea, overview, and references about Medical Supply Chain Management (e.g. overview document, research/reference material).                                                                                                                                      |
| **docs**   | MVP documentation         | Document set for the MVP: Executive Summary, PRD, Technical Architecture, Smart Contract Logic, Data Model, UI/UX Wireframes, Non-Functional Requirements, Timeline & Milestones, Legal/Commercial, Risk & Assumptions. See `docs/README.md` for the full index. |
| **code**   | MVP templates and samples | Template MVP projects and sample implementations (e.g. Fabric chaincode, REST API, frontend) to bootstrap development.                                                                                                                                           |

### Quick reference

- **master** — Start here for context: what the project is, why blockchain, and key references.
- **docs** — Use this branch when sharing or working with formal MVP docs (e.g. with clients or development teams).
- **code** — Use this branch to get or extend template/sample code for the MVP.

---

## Getting started

1. Clone the repository and list branches:  
   `git branch -a`
2. Switch to the branch you need, for example:  
   `git checkout master` | `git checkout docs` | `git checkout code`
3. On **docs**, open `docs/README.md` for the documentation index and links to each MVP document.
